SRVE

A full-stack sports community platform built to connect players, organize matches, and grow active sporting ecosystems. At its core is an automated matchmaking engine that dynamically pairs players into Single or Double formats based on sport category and group capacity — no manual coordination needed.

Backed by a RESTful API for seamless state sync between the Django backend and a cross-platform Flutter mobile app. CI/CD pipelines via GitHub Actions and Bitrise keep deployments fast and reliable.
